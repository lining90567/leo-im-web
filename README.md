## 运行环境要求

- Chrome、IE10+

## 主要功能

- 私聊
- 群聊
- 文字、表情、图片、文件

## 配置

- 开发环境
	config/dev.env.js
	
	设置BASE_API与WEBSOCKET_URL
	
- 生产环境
	config/prod.env.js
	
	设置BASE_API与WEBSOCKET_URL	
	
## 安装依赖包

- 首次启动之前需要安装依赖包	npm install（建议使用cnpm）

## 本地运行

- npm run dev

## 构建

- npm run build
